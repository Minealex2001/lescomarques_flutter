Flutter - widgets

Abiertos: dijous, 4 de gener de 2024, 00:00
Pendiente: dimecres, 31 de gener de 2024, 23:59

Para esta labor crearemos varias pantallas para una futura aplicación que proporcione información sobre las comarcas de la Comunitat.

Por el momento, no añadiremos ninguna funcionalidad, sólo el diseño de las pantallas para mostrar la información. Para ello, se le proporciona un archivo comarcas.dart con un Map con la información sobre las provincias y las comarcas de la Comunidad, por si desea acceder directamente a esta información.

Las pantallas a diseñar serán las siguientes:

1.1 Pantalla de inicio

Una pantalla típica de login, que constará de un logotipo (del instituto Álvaro Falomir), un texto con el título de la aplicación, y dos campos de texto, para el usuario y la contraseña.

En la siguiente unidad abordaremos la creación de formularios. Por el momento, el widget que necesitaremos para añadir un campo de texto es TextFormField, y puede utilizarlo con el siguiente código:

TextFormField(
    decoration: const InputDecoration(
        filled: true,
        hijoColor: Colors.white,
        labelText: 'Usuario',
        border: OutlineInputBorder(),
    ),
),

Con la propiedad decoration podemos asignar un estilo al campo de texto, en cuyo caso, le debemos añadir el borde, un color de fondo y la etiqueta Usuario.

1.2 Pantalla Provincias

Esta pantalla mostrará las tres provincias. Puede obtener la información sobre ellas (nombre e imagen representativa) a partir del archivo comarcas.dart que se le proporciona.

Si importa el archivo, para acceder, por ejemplo a la imagen representativa de la provincia de Valenia (que se encuentra la primera), lo haríamos con: provincias["provincies"][0]["img"], para la d Alicante con provincias["provincies"][1]["img"] y para Castellón provincias["provincies"][2]["img"].

En esta pantalla se mostrará una lista con las comarcas de alguna de las provincias. Podemos hacer uso de Cards que combinan imagen y nombre de la comarca, o cualquier otro tipo de widget. La información que contendrá deberá ser básicamente el nombre de la comarca y la imagen representativa.

1.4 Pantallas con información sobre la comarca

Por último, generaremos dos pantallas con información sobre una comarca en concreto (de momento, esta comarca será fija).

En estas dos pantallas mostraremos la información que se encuentra para una comarca en concreto en el fichero de comarcas: comarca, capital, población, imagen representativa (img), descripción (desc), y latitud y longitud (vector coordenadas).

Además, también maquetaremos un espacio en una de ellas para añadir una imagen sobre el tiempo (de momento será fija), así como la temperatura actual y la dirección del viento.
